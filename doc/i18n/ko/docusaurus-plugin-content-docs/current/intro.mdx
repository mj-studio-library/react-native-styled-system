# 시작하기

`React Native Styled System`은 Web에서 흔히 사용되는 `styled-system`패키지의 React Native용 구현입니다.

:::info

완전히 `styled-system`의 문법을 지원하지 않고 현재 지원하지 않는 문법도 있지만 이는 기술적 한계가 아니며 필요에 의해 추가될 예정입니다.

React Native에서 아직 지원되지 않는 `justifySelf` 같은 스타일들은 당연히 추후에도 추가될 수 없습니다.
:::

## 지원되는 기능

- `styled-system`의 문법대로 `m, px, py, bg, flex, flexDirection, position`등의 인자들을 Props로 바로 전달할 수 있게 합니다.
- 사용자가 직접 테마를 정의하고 전달하여 디자인 시스템을 정의할 수 있습니다.
- CLI를 이용한 Type Generation과정으로 TypeScript를 완전히 이용할 수 있습니다.
- `safeAreaTop`, `sidePadding`같은 논리적이거나 반응형으로 변경되는 값들을 테마에 주입시켜 토큰값으로 활용할 수 있습니다.

## 한계

- 현재 지원하는 토큰들은 `space`, `sizes`, `colors`뿐이며 추후에 `fontSize`, `zIndices`같은 토큰을 추가할 예정입니다.
- 모든 `View`의 스타일 필드나 Prop들을 아직 지원하지 않습니다.
- 아직 다크모드 기능은 존재하지 않습니다.
